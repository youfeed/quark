# Card 卡片

### 介绍

用于组织信息和操作，通常也作为详细信息的入口。

## 代码演示

```html
<cs-card
    title='2021-02-08 12:00 | 单车'
    content='2分50秒'
    cash='1元' 
    desc='已完成'
></cs-card>
```


## API

### Props

| 参数         | 说明                             | 类型   | 默认值           |
|--------------|----------------------------------|--------|------------------|
| title        | 标题 | String                     |
| content      | 内容 | String                     |
| cash         | 金额 | String/Number                     |      0       |
| desc         | 状态 | String                     |         |
